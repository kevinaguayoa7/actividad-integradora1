PROYECTO: Aplicación Flutter Interactiva
AUTOR: Kevin Aguayo

Proyecto Flutter académico con navegación, SnackBar y widget adicional.



DESCRIPCIÓN

Este proyecto corresponde a una aplicación Flutter básica desarrollada

para demostrar el uso de navegación, interacción con botones y análisis

correcto del código sin errores.



FUNCIONALIDADES

\- Modificación del título de la aplicación.

\- Cambio de color del AppBar en ambas pantallas.

\- Texto principal personalizado.

\- Botón con mensaje SnackBar.

\- Navegación entre pantallas usando Navigator.push y Navigator.pop.

\- Implementación del widget ExpansionTile para mostrar información adicional.



VALIDACIÓN

El proyecto fue analizado correctamente utilizando:

\- flutter analyze (sin errores)

\- flutter doctor (sin errores críticos)
